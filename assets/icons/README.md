# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (for extension icon in Chrome menu)
- `icon48.png` - 48x48 pixels (for extension management page)
- `icon128.png` - 128x128 pixels (for Chrome Web Store)

**Required for extension to load properly.**

You can create placeholder icons or use icon generation tools:
- Use a tool like [Favicon Generator](https://favicon.io/)
- Or create simple PNG files with the extension name/logo
